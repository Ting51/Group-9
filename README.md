# 主題：線上穿搭服飾商店

### 功能：
* 整合各式電子商店平台商品，像是MUJI、GU等商店，提供使用者一個完整的商品瀏覽介面

* 根據使用者的商品去和VR做結合，提供線上穿搭的功能，減少消費者購物後的尺寸或是風格不搭等問題
